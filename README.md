# Fokir - Personal Portfolio

Fokir is a modern, responsive one-page portfolio template designed for professionals. It’s built with a focus on clean code and elegant UI/UX.

## 🛠️ Built With
- **HTML5:** For semantic and accessible structure.
- **CSS3:** Custom animations, transitions, and responsive grid layouts.
- **Font Awesome:** For professional iconography.
- **Google Fonts:** Integrated for high-quality typography.

## 🌟 Features
- **Smooth Scrolling:** Enhanced navigation between sections.
- **Filterable Portfolio:** Showcasing work across different categories.
- **Contact Form:** A styled, user-friendly communication section.
- **Responsive Navigation:** A mobile-ready menu for smaller viewports.

## 📐 Development Goal
The primary objective of this project was to master the art of building a "Professional Grade" UI from scratch, focusing on details like spacing, consistency, and cross-browser compatibility.
